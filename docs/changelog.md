# Loving 更新日志

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-17

### 🎉 首发版本

#### 新增功能

- **飞行棋游戏**
  - 四种版本模式（普通版/情侣版/热恋版/夫妻版）
  - 16格棋盘 + 起点终点
  - 真实骰子动画效果
  - 每格不同任务提示
  - 玩家头像标识（男神/女神）
  - SVG渐变图标设计
  - 完成任务切换机制

- **亲密卡牌**
  - 5种版本（恋爱版/热恋版/同居版/进阶版/私密版）
  - 3D翻转卡片动画
  - VIP锁定标识
  - 随机卡牌编号

- **姿势大全**
  - 版本选择标签
  - 难度等级（入门/进阶/高级）
  - VIP锁定显示
  - 玩法说明区域

- **首页优化**
  - Hero区域 + badges
  - 游戏卡片悬停效果
  - 标签样式分类（热门/新功能/VIP）
  - 功能特性展示区

- **主题系统**
  - 四种主题（浪漫/简约/活泼/高端）
  - ThemeSwitcher组件
  - VIP试用次数限制
  - localStorage持久化

- **登录系统**
  - 手机号+验证码登录
  - 内置管理员账户（13800138000）
  - VIP自动判定

#### 技术改进

- Vue 3.4 + TypeScript
- Vite 5 快速构建
- Pinia 状态管理
- Vue Router 4 路由
- Tailwind CSS 3 样式
- CSS Variables 主题系统

#### 文档新增

- README.md 项目说明
- docs/technical-design.md 技术设计文档
- docs/user-guide.md 用户使用指南
