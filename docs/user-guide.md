## 二、游戏介绍

### 2.1 飞行棋 🎲

飞行棋是平台的核心游戏，支持双人同屏对战。

**游戏入口：** 首页 → 飞行棋卡片

#### 版本说明

| 版本 | 描述 | 适合人群 |
|------|------|----------|
| 普通版 💫 | 牵手、拥抱、情话等基础任务 | 刚在一起的情侣 |
| 情侣版 💑 | 十指紧扣、法式亲吻等浪漫任务 | 热恋期情侣 |
| 热恋版 🔥 | 激吻、咬耳朵等刺激任务 | 感情稳定的情侣 |
| 夫妻版 💍 | 回忆初见、蜜月时光等深度任务 | 已婚夫妻 |

#### 游戏规则

1. **开始游戏**
   - 选择一个版本（普通/情侣/热恋/夫妻）
   - 默认玩家1先出手

2. **投掷骰子**
   - 点击"投掷骰子"按钮
   - 骰子会显示动画效果
   - 最终显示点数（1-6）

3. **移动棋子**
   - 骰子点数决定移动步数
   - 棋子自动移动到新位置

4. **完成任务**
   - 落在格子上显示任务内容
   - 玩家需完成该任务
   - 点击"完成任务"继续

5. **胜利条件**
   - 最先到达终点的玩家获胜
   - 可以选择"再来一局"

## 五、常见问题

### 5.1 登录问题

**Q: 收不到验证码怎么办？**
A: 内置账户（13800138000）可以直接登录，无需验证码。

**Q: 登录后显示未登录？**
A: 尝试刷新页面，或点击"内置账户登录"重新登录。

### 5.2 游戏问题

**Q: 飞行棋骰子不动？**
A: 检查网络连接，尝试刷新页面。

**Q: 卡牌无法翻转？**
A: 确保使用的是较新版本的浏览器。

**Q: 任务无法完成？**
A: 可以选择跳过或认输，系统会自动切换到对方。

### 5.3 主题问题

**Q: 主题切换后不生效？**
A: 某些主题需要VIP权限，试用次数用完需要开通VIP。
